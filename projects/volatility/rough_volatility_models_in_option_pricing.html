Rough Volatility Models
in Option Pricing

Fractional Brownian motion · Rough Heston model · ISI Kolkata

GitHub Repository →

Motivation

Classical stochastic volatility models — from Black–Scholes to Heston — assume that volatility evolves as a semimartingale. Empirical study of high-frequency realised volatility time series reveals, however, that volatility paths have a Hurst exponent H significantly below ½, meaning they are rougher than Brownian motion.

This project explores the implications of modelling volatility via fractional Brownian motion with H ≈ 0.1, following the programme initiated by Gatheral, Jaisson, and Rosenbaum (2018).

Models Studied

  • Rough Bergomi model — variance driven by the Riemann–Liouville fractional Brownian motion kernel; forward variance parametrisation.
  • Rough Heston model — fractional integral version of the classical Heston SDE; admits semi-analytical option pricing via characteristic function methods.
  • Baseline comparison — standard Heston and SABR models as benchmarks for implied volatility surface fit quality.

Methods

Option prices are computed by Monte Carlo simulation of fractional Brownian paths using Cholesky decomposition of the covariance matrix and the hybrid scheme of Bennedsen, Lunde, and Pakkanen. Implied volatility surfaces are calibrated to SPX option data and compared across models.

Status

This is an ongoing independent study project. Code and preliminary findings are available in the GitHub repository.
